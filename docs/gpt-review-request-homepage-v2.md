# Magic Compass 首頁 V2 精修審稿交接

本輪為 **Vue 首頁靜態 UI 第二輪精修**，未更改首頁模組順序，仍維持：

Header -> Hero Banner -> 核心數據條 -> 熱門市場報價 -> 平台介紹 / 核心賣點 -> 核心優勢區 -> 平台下載 / 終端支持 -> 簡版即時報價表 -> 品牌視頻 + 開戶流程 -> 信任背書 / 品牌實力 -> 底部 CTA -> Footer

2. 核心數據條改得更像正式金融資訊條
3. 熱門市場卡片、tab、間距與層級細修
4. 平台介紹區改成更明確的左文右圖主次
5. 下載區資訊節奏重新整理
6. 即時報價表桌機版壓縮密度，手機版改為卡片式
7. 視頻區、開戶流程、信任背書與 Footer 收尾加強
