Preview Methods

Method A: Direct Open
1. Extract the zip package.
2. Double-click index.html.
3. This package has been prepared as a file-friendly static snapshot.
4. Direct open has been locally validated.

Method B: One-Click Local Preview
1. Extract the zip package.
2. Double-click start-preview.bat.
3. Your browser should open:
   http://127.0.0.1:4173/
4. Keep the preview window open while reviewing.
5. Close the PowerShell window to stop the local preview server.

Recommendation:
- Direct open is supported.
- Use start-preview.bat if you prefer a local server preview.
